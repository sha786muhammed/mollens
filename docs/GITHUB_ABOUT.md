## Short Description

MolLens is an open-source molecular workbench for SMILES, image-based structure recognition, 2D editing, and interactive 3D visualization.

## Tagline Alternatives

- Open-source chemistry workbench for 2D sketching and 3D molecular visualization
- Generate, edit, inspect, and export molecules from SMILES, names, and images
- Browser-based molecular editor and RDKit-powered 3D visualization platform
